# Cloudflare DDNS Update
## Cách sử dụng
##### Bước 1: Tải file về máy
```` 
wget https://t.ly/jd55t
````
##### Bước 2: Cấp cho file có quyền thực thi
````
chmod +x update.sh 
````
##### Bước 3: Chạy file
````
./update.sh
````
### Lưu ý: Trước khi chạy hãy nhớ chỉnh lại các phần như zone id, domain,v.v nhé. Chỉnh lại bằng lệnh
````
nano update.sh 
````
### Hoặc 
````
vim update.sh
````
